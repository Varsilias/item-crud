## Description

[ITEM CRUD](https://github.com/nestjs/nest) a mini project to showcase the awesomeness of NestJs.

## Installation

```bash
$ npm install or yarn install (If you are using yarn)
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
